# PriceSentry

PriceSentry — это веб-приложение для отслеживания изменения цен на товары в интернет-магазинах. Проект состоит из бэкенда на ASP.NET Core и фронтенда на TypeScript. Основная цель — получать уведомления в Telegram, когда цена на интересующий товар достигает желаемого уровня.

## Функциональность

*   **Отслеживание цен:** Добавление ссылок на товары для мониторинга.
*   **Настройка триггеров:** Установка целевой цены, при достижении которой приходит уведомление.
*   **Интеграция с Telegram:** Привязка аккаунта Telegram для получения оповещений.
*   **Управление списком:** Возможность просматривать, добавлять и удалять отслеживаемые товары.

## Стек технологий

*   **Бэкенд:** C#, ASP.NET Core, Entity Framework Core, SQLite
*   **Фронтенд:** TypeScript, HTML, CSS
*   **Интеграции:** Telegram Bot API
*   **Инструменты:** Git

## Установка и запуск

### Требования

*   [.NET SDK](https://dotnet.microsoft.com/download) (версия 8 или 9)
*   [Node.js](https://nodejs.org/) (для фронтенда)
*   Git

### Инструкция по запуску (локально)

1.  **Клонировать репозиторий**
      ```bash
      git clone https://github.com/IlyaRx/PriceSentry.git
      cd PriceSentry
      ```
    
2.  **Настройка и запуск бэкенда**
      ```bash
      cd PriceSentry.Beckend
      dotnet restore
      dotnet run
      Бэкенд будет доступен по адресу https://localhost:5001 или http://localhost:5000.
      ```

3.  **Настройка и запуск фронтенда**
Откройте новый терминал в корне проекта:
      ```bash
      cd PriceSentry.Frontend
      # Если есть package.json
      npm install
      npm start
      # Или просто откройте index.html в браузере
      ```
      
4. Настройка Telegram-бота

*   Создайте бота через @BotFather и получите токен.
*   Добавьте токен в конфигурацию бэкенда (например, в appsettings.json).
